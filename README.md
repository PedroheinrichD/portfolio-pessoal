# Portfólio Pessoal

Portfólio desenvolvido como trabalho da faculdade para apresentar minhas habilidades, tecnologias estudadas e projetos desenvolvidos durante minha jornada em Análise e Desenvolvimento de Sistemas.

O projeto foi construído com foco em uma interface moderna, responsiva e organizada, funcionando como uma vitrine profissional dos meus estudos e evolução como desenvolvedor Front-end.

# Funcionalidades

* Apresentação pessoal
* Seção "Sobre mim"
* Exibição das principais habilidades
* Cards com projetos desenvolvidos
* Link para visualização dos projetos
* Link para o código-fonte dos projetos
* Informações de contato
* Links para GitHub, LinkedIn e Instagram
* Layout responsivo para desktop, tablet e mobile

# Destaques técnicos

## Estrutura semântica

O projeto foi desenvolvido utilizando HTML semântico para melhorar a organização do código e facilitar a manutenção.

---

## Design responsivo

Toda a interface foi adaptada para diferentes tamanhos de tela utilizando Media Queries, garantindo uma boa experiência em dispositivos móveis e desktops.

---

## Organização do CSS

Os estilos foram organizados por seções da página, facilitando futuras manutenções e expansões do projeto.

---

## Interface moderna

O layout utiliza uma paleta em tons escuros com detalhes em roxo, animações suaves e organização visual voltada para uma apresentação profissional.

---

# Objetivo do projeto

Este projeto teve como objetivo praticar:

* Estruturação de páginas HTML
* Estilização com CSS
* Responsividade
* Organização de componentes visuais
* Boas práticas de desenvolvimento Front-end
* Versionamento com Git e GitHub
